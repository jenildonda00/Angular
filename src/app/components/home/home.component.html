  2. The Open Library API is a RESTful web service to obtain book information. It aims to create one web page for every book ever published, a lofty but achievable goal.

    To build Open Library, we need hundreds of millions of book records, a wiki interface, and lots of people willing to contribute their time and effort. So far, we've gathered over 20 million records from various large catalogs and single contributions, with more on the way.
